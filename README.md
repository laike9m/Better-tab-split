# Better Tab Split

A Chrome extension that opens links in the opposite pane when in Split View, instead of a new tab.

一个 Chrome 扩展，让你在拆分视图（一个 tab 左右分屏双开页面）中点击链接时，在另一侧打开而不是新标签页。

![](store_assets/promo_small_440x280.jpeg)

## Installation

[![Available in the Chrome Web Store](docs/chrome_web_store_badge.png)](https://chromewebstore.google.com/detail/better-tab-split/epcgalejpnmaoiccdkocnkolochbgegg)

## Prerequisites

Make sure you have Chrome 142 or higher and enabled Split View feature.

![alt text](docs/split_view_open_zh.png)

## Icon Status

Just click the icon to toggle the extension on and off.

| Icon | Status | Description |
|:----:|--------|-------------|
| ![Disabled](icons/icon-disabled-32.png) | Disabled | Current tab is not using Split View |
| ![Off](icons/icon-off-32.png) | Off | Split View is enabled, but extension is off |
| ![On](icons/icon-on-32.png) | On | Split View is enabled and extension is active |

点击图标即可切换扩展功能的开启/关闭状态。

| 图标 | 状态 | 说明 |
|:----:|------|------|
| ![禁用](icons/icon-disabled-32.png) | 禁用 | 当前标签页未启用拆分视图 |
| ![关闭](icons/icon-off-32.png) | 关闭 | 拆分视图已启用，但扩展功能关闭 |
| ![开启](icons/icon-on-32.png) | 开启 | 拆分视图已启用，扩展功能开启 |
